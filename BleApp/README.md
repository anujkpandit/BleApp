# BleApp - BLE Scanner

This is a React Native application built with Expo that demonstrates how to use `react-native-ble-plx` to scan for and interact with Bluetooth Low Energy (BLE) devices.

## âœ¨ Features

- **BLE Device Scanning**: Discover nearby Bluetooth Low Energy devices.
- **Cross-Platform**: Runs on Android and iOS using a single codebase.
- **Modern Tech Stack**: Built with Expo, TypeScript, and Expo Router for file-based routing.
- **Themed UI**: Supports both light and dark modes.

## ğŸš€ Getting Started

Follow these instructions to get the project up and running on your local machine for development and testing.

### Prerequisites

- Node.js (LTS version recommended)
- npm or yarn
- Expo Go app on your physical device (for the easiest setup)
- Android Studio or Xcode for running on emulators/simulators.

### Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/anujkpandit/BleApp.git
    cd BleApp
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Permissions Setup (Crucial for BLE)**

    For Bluetooth functionality to work, you need to add the necessary permissions to the native project files.

    #### Android

    Ensure the following permissions are present in your `android/app/src/main/AndroidManifest.xml`:

    ```xml
    <!-- Required for BLE on API 31+ -->
    <uses-permission android:name="android.permission.BLUETOOTH_SCAN" />
    <uses-permission android:name="android.permission.BLUETOOTH_CONNECT" />

    <!-- Required for BLE on API 23-30 -->
    <uses-permission android:name="android.permission.BLUETOOTH" />
    <uses-permission android:name="android.permission.BLUETOOTH_ADMIN" />
    <uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />

    <!-- Add this feature to your manifest to indicate that your app uses BLE -->
    <uses-feature android:name="android.hardware.bluetooth_le" android:required="true"/>
    ```

    #### iOS

    Ensure the following keys are present in your `ios/BleApp/Info.plist`:

    ```xml
    <key>NSBluetoothPeripheralUsageDescription</key>
    <string>This app uses Bluetooth to find and connect to nearby devices.</string>
    <key>NSBluetoothAlwaysUsageDescription</key>
    <string>This app uses Bluetooth in the background to stay connected to your devices.</string>
    ```

### Running the Application

There are two main ways to run the app:

1.  **Using the Expo Go app (recommended for quick start):**

    This method allows you to run the app on a physical device without needing to build the native code.

    ```bash
    npm start
    ```
    Then, scan the QR code with the Expo Go app on your Android or iOS device.

2.  **Using a Development Build:**

    This method compiles the native code and is necessary for testing native features like BLE on simulators or for creating a standalone build.

    - **For Android:**
      ```bash
      npm run android
      ```

    - **For iOS:**
      ```bash
      npm run ios
      ```
